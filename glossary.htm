Кинологический словарь
Кинология
Наука о собаках, выведении пород собак, дрессировке собак, истории происхождения собак.
Собака
Домашнее животное, одно из наиболее распространённых животных-компаньонов.
Аллюр
Способ движения животных, в том числе собак. Различают шаг, иноходь, рысь, галоп, карьер.
Выставка собак
Зоотехническое мероприятие, при котором собак оценивают в зависимости от соответствия их экстерьера стандарту породы, сравнивают между собой для выявления лучших особей - носителей современного типа породы, наиболее ценных для разведения.
Генотип
Совокупность всех генов организма, локализованных в хромосомах, его наследственная материальная основа.
